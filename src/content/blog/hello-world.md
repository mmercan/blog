Welcome to my blog! This is my first post.

## What to expect

I'll be writing about things I find interesting — projects, ideas, and whatever else
is on my mind.

## How this works

This blog is built with [Astro](https://astro.build), stored entirely in a GitHub repo,
and deployed automatically to GitHub Pages every time I push to `main`.

Posts are just Markdown files. No CMS, no database, no monthly bill.
